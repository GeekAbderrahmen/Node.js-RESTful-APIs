# Node.js RESTful APIs
Node.js RESTful APIs build with ExpressJS, NodeJS and MongoDB

## Install
```sh
npm install
```

## Launch Mongod
```sh
mongod
```

## Run
```sh
npm start
```

## Test
Test the APIs on a REST client (e.g: POSTMAN)
> [http://localhost:3000/api](https://codepen.io/GeekAbderrahmen/full/WOeGXE/)
